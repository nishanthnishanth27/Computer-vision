# 🚀 AI Virtual Mouse Controller
Control your computer cursor like **Iron Man** using Hand Gestures! 🤖💻

## ✨ What is this?
This is an Artificial Intelligence project that uses **Computer Vision** to track your hand movements through a webcam and converts them into mouse actions. No physical mouse needed!

## 🛠️ How I built it:
- **Language:** Python
- **Libraries:** OpenCV, Mediapipe, PyAutoGUI
- **Concept:** Hand Landmark Detection (21 points tracking)

## ⚡ Features:
- 🖐️ **Smooth Tracking:** Move your index finger to move the cursor.
- 🖱️ **Virtual Click:** Pinch your fingers to perform a mouse click.
- 🚫 **Zero Hardware:** Works with any standard laptop camera.
